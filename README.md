# PAHStack
### A remake of Stack Overflow



## Team project created using ASP.NET and Entity Framework.
    -Entity Framework 
    -Javascript (jQuery and Ajax)
    -ASP.NET Razor
    -MS SQL Server 
    -OOP and Functional programming
    -Full use of the MVC concept in ASP.NET
    -Workflow with a team
